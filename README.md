# Python-Technical-Delivery-Project
Population in Europe and Africa

A data analysis project completed as part of a Data Analyst application for GlobalTech, using Pandas for data manipulation and Matplotlib for visualisation. The goal was to analyse global population statistics, extract insights, and present findings through code, charts and summary outputs.

Tech Stack

Python | Pandas | Matplotlib | NumPy | Jupyter Notebook

Project Highlights

✔ Loaded, cleaned and reshaped the dataset for analysis
✔ Performed population comparisons across regions and years
✔ Created visual insights using bar charts and line plots
✔ Delivered results for real analytical questions

Tasks Completed

Identified countries with 0 population recorded in 2000

Calculated total population in Africa (2010) + bar chart visual

Found average population of South America (2000) & compared above/below average countries

Highlighted countries exceeding 1B population in 2007 + plot visualisation

Calculated population growth in Europe (2000–2010) + line chart for top 5 countries

Outcomes

Visualised population distribution across regions and years

Extracted key insights using grouping, filtering and aggregation

Demonstrated analytical thinking using real-world style data questions

Summary

This project showcases my ability to clean, analyse and visualise data, turning raw information into insights — reflecting the technical skills required for an entry-level Data Analyst role.
